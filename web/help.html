Help & Support – User Guide

1️⃣ System Login

  • Login using your assigned username and password.
  • After successful login, you will be redirected to the Dashboard.
  • You can return to the Dashboard from any page using the Back to Dashboard button.

2️⃣ Guest Registration

  • Navigate to Guest Registration from the Dashboard.
  • Enter Full Name, Address, Contact Number, and NIC.
  • Click Register Guest to save the guest details.
  • A confirmation message will be displayed upon successful registration.

3️⃣ Book a Room

Loading Guest Details

  • Enter the guest NIC and click Load.
  • If registered, guest details will auto-fill.
  • If not registered, return to Guest Registration.

Checking Availability

  • Select a Room Type.
  • Choose Check-In and Check-Out dates.
  • Click Check Availability.
  • Available rooms will appear in the dropdown list.

Booking Confirmation

  • Select a room and click Book Room.
  • A confirmation message will display Reservation ID and Total Amount.

4️⃣ Guest Check-Out

  • Filter reservations using From and To dates.
  • Reservations will appear in a table.
  • Enter Reservation ID and click Process Check-Out.
  • Status will change to CHECKED_OUT and room becomes AVAILABLE.
  • Click Available Rooms To navigate Available rooms page and display the ACTIVE & AVAILABLE rooms on given dates

5️⃣ Calculate & Print Bill

View Active Reservations

  • Enter NIC to load active reservations.
  • Table displays Reservation ID, Dates, Nights, and Amount.

Generate & Print Bill

  • Enter Reservation ID and click Generate.
  • Printable invoice will display Reservation No, NIC, Name, Room Number, Dates, Nights, and Total Amount.
  • Click Print Bill to print the invoice.
